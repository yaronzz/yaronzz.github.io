将代码提交到Github时，突然报错：

这是由于Github提高了仓库的安全性，访问仓库时需要提供令牌token

生成token

  1. 访问Personal Access Tokens (github.com) ，点击生成新token

  1. 配置名称、有效期、访问权限

  1. 点击复制token，此token只能查看一次，最好自己保存到其他地方
